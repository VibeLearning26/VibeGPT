# VibeGPT — How to Run the Website

## Prerequisites
- Docker Desktop installed and running
- Python 3.11–3.13 (backend)
- Node.js (frontend)

---

## 1. Start the database + LLM (Docker)

```powershell
cd infrastructure
docker compose --env-file ../.env up postgres ollama -d
```

Wait until both containers are healthy (`docker ps`).

First time only — pull the LLM model:
```powershell
docker exec vibegpt-ollama ollama pull llama3.2:3b
```

---

## 2. Start the backend API (port 8000)

```powershell
cd services/api
.venv\Scripts\activate
uvicorn app.main:app --port 8000
```

Leave this running.

---

## 3. Start the document worker

This indexes uploaded files. It must run alongside the API.

```powershell
cd services/api
.venv\Scripts\activate
python -m app.workers.document_worker
```

Leave this running.

---

## 4. Start the frontend (port 3000)

```powershell
cd apps/web
npm run dev
```

---

## 5. Open the app

Go to http://localhost:3000

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Site won't load | Make sure all 4 steps above are running |
| Login fails | Confirm the API (step 2) is running on port 8000 |
| Uploads stuck at "processing" | Start the document worker (step 3) |
| Docker containers not starting | Open Docker Desktop; check `docker ps` |
| AI answers not working | Pull the model in step 1; confirm Ollama container is up |
